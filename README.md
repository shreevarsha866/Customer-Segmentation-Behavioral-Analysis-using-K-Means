# Customer Segmentation & Behavioral Analysis using K-Means

## 📌 Project Overview
This project applies K-Means clustering to segment customers based on demographic and behavioral attributes. The objective is to identify distinct customer groups and provide insights that enable targeted marketing and personalized business strategies.

---

## 🎯 Objectives
- Perform data inspection and preprocessing
- Apply Exploratory Data Analysis (EDA) to understand customer behavior
- Segment customers into meaningful clusters using K-Means
- Visualize and interpret customer segments for business insights

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Power BI

---

## 📊 Key Steps Performed
### 1. Data Inspection & Preprocessing
- Reviewed dataset structure and feature distributions
- Handled missing values and outliers
- Scaled numerical features for clustering

### 2. Exploratory Data Analysis (EDA)
- Analyzed customer demographics and income patterns
- Studied relationships between age, income, and spending behavior

### 3. K-Means Clustering
- Applied K-Means algorithm to segment customers
- Identified **4 distinct customer segments**
- Evaluated cluster distribution and characteristics

### 4. Visualization & Insights
- Created visualizations to compare customer segments
- Built interactive dashboards in Power BI
- Identified income- and age-based behavioral patterns

---

## 📈 Key Insights
- Customers were grouped into **4 distinct segments**
- Improved customer targeting efficiency by approximately **20%**
- Identified high-value and budget-conscious customer groups
- Enabled data-driven personalized marketing strategies
